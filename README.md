# Interior Decoration AI

Projeto de decoração de interiores com IA, criado com Next.js + TypeScript + Tailwind CSS.

## Tecnologias

- **Next.js 15** com App Router
- **TypeScript**
- **Tailwind CSS v4**
- **shadcn/ui** para componentes
- **Lucide React** para ícones
- Suporte a múltiplos idiomas (PT, EN, ES, FR, DE, IT)

## Como rodar localmente

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador.

## Deploy

Este projeto está configurado para deploy no **Vercel**.

1. Conecte o repositório no [Vercel](https://vercel.com)
2. O deploy é automático a cada push no `main`
3. Framework: Next.js (detectado automaticamente)
